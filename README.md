# Shaman.WinForms.Windows7Renderer

Renders ToolStrip

## Usage
```csharp
using Shaman.WinForms;

ToolStripManager.Renderer = Windows7Renderer.Instance; // Main theme features
Windows7Renderer.Initialize(toolstrip1); // Font and other details
```
